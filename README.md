# Expense Tracker

A full-stack Expense Tracker application built using FastAPI, SQLite, HTML, CSS, JWT Authentication, and Bcrypt Password Hashing.

## Features

### Authentication

* User Registration
* User Login
* JWT Token Authentication
* Secure Password Hashing using Bcrypt

### Expense Management

* Add Expenses
* View Expenses
* Edit Expenses
* Delete Expenses

### Dashboard

* Expense Overview
* Total Transactions
* Monthly Expense Reports
* Yearly Expense Reports

### User Management

* Profile Section
* Delete Account

## Technologies Used

### Backend

* Python
* FastAPI
* SQLite
* JWT Authentication

### Frontend

## Installation

Clone the repository:

git clone https://github.com/anilkumar234-dot/Expense-Tracker.git

Navigate to the project directory:

cd Expense-Tracker

Install dependencies:

pip install -r requirements.txt

Run the application:

uvicorn expense:app --reload

Open in browser:

http://127.0.0.1:8000
